## Panduan Instalasi
1. Clone Repository

git clone https://github.com/kayyerl-nath/Kelompok-Bebas-SNEIK.git
cd Kelompok-Bebas-SNEIK

2. Instal library bila diperlukan

Jika game menggunakan library tambahan (misal curses, pygame, dll.), instal dengan:

pip install -r requirements.txt
(Jika tidak ada requirements, bagian ini dapat dilewati.)
